> ## Description 
> A social media platform called HobbyCircles connects users with online and local groups based around shared hobbies.  HobbyCircles makes it simple to find active groups, get involved with events, and create connections, no matter what your interests is could be hiking, board games, book clubs, or coding.

> Hobby Circles is made to make it easier for people to find and connect with others who share their interests and hobbies. It solves the problem of hobby groups and community activites being scattered across multiple apps and websites, making them difficult to find and organize.


## App Functions
1. Customer (the user with the customer role):
   1. Create/modify customer profile — Users sign up and fill out a profile with their name, photo, and hobbies.
   2. View available services — Customers look/search for groups by interest or location.
   3. Subscribe to available services — Customers join groups and RSVP to events.
   4. Write reviews for subscribed services — Customers leave ratings and reviews for groups they've joined.
  
       
2. Provider (the user with the provider role):
    1. Create/modify/remove organizer profile — customers set up a profile with their name, bio, and interests.
    2. Create services — organizer create group pages, add events to a calendar, and manage group details.
    3. View customer statistics — organizer see how many members and RSVPs their groups have.
    4. Reply to reviews — organizers respond to reviews left on their pages.
  
2. Provider (User with the provider role):
    1. Group creators can create a provider profile that repersents them as a community host.
    2. Providers can create groups and events, including schedule, descriptions, set rules, etc.
    3. Providers can view statistics based on how many attendees and engagement rates.
    4. Providers can respond back to users reviews in order to improve group events and engage with the community. 

3. SysAdmin (the user with the admin role if applicable):
    1. Manage user access — Admins can ban or restore user accounts or check activity.
    2. Moderate services — Admins can remove groups or events that break the rules.
    3. Moderate reviews — Admins can delete inappropriate reviews.
    4. View usage statistics — Admins see platform-wide data like total users, groups, and events.
